Como Usar o Emaulador

  1. Clique na área "Carregar ROM" ou arraste um arquivo .bin para a área destacada.
  2. Após carregar a ROM, clique em "Iniciar" para começar a emulação.
  3. Use as teclas indicadas para controlar o jogo (Setas, Z, X, Enter, Espaço).
  4. Use os botões "Pausar", "Reiniciar" e "Parar" para controlar a execução.
  5. Para uma experiência imersiva, use o botão "Tela Cheia".

Nota: Este emulador é compatível com arquivos binários (.bin) de sistemas inspirados no Emanuel.
